# 安装指南 - Chrome 浏览器

1. 前往 [chrome 网上应用店
StarMask 下载页](https://chrome.google.com/webstore/detail/starmask/mfhbebgoclkghebffdldpobeajmbecfk/related)

2. 点击 `添加至 Chrome`

    ![](./images/1.jpg)

3. 点击 `开始使用`

    ![](./images/2.jpg)

4. 如果是第一次安装使用，请点击 `创建钱包`

    ![](./images/3.jpg)

5. 点击 `我同意`

    ![](./images/4.jpg)

6. 输入密码，点击 `创建`

    ![](./images/5.jpg)

7. `点击此处显示密语`

    ![](./images/6.jpg)

> 提示:
> - 下载助记词，方便以后通告导入助记词的方式恢复钱包
> - 可以选择 `稍后提醒` 直接跳到 第8步， 以后再根据提示进行备份

8. 点击 `下一步`

    ![](./images/7.jpg)

9. 按顺序输入助记词， 点击 `确认`

    ![](./images/8.jpg)

10. 点击 `全部完成`

    ![](./images/9.jpg)

11. 进入插件钱包主页面(全屏模式)

    ![](./images/10.jpg)
 
12. 点击浏览器右上角`扩展程序`图标，选中 `StarMask` 右侧 图钉

    ![](./images/11.jpg)

13. StarMask插件的图标会固定出现在浏览器右上角

    ![](./images/12.jpg)

14. 点击 StarMask插件的图标，弹出插件钱包主页面(弹窗模式)

    ![](./images/26.jpg)
